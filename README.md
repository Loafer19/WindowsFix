# Windows Fix

![logo](./ServicesManager/web/public/logo.png)

Run several scripts to improve the system

## Services Manager

Will show useful information about services on your PC

to start exploiting

- in `server` directory
  - run `bun i`
  - then `bun .\app.js`
- in `web` directory
  - run `bun i`
  - then `bun run dev`
  - open in browser
